Cover&Case Creator

📖 Sobre o Projeto
* O CustomCase é um software que permite aos usuários criar capas de celular personalizadas de forma simples e intuitiva.
* O sistema oferece ferramentas para adicionar imagens, textos, cores, desenhos e visualizar a capa em um modelo real de celular antes da produção.
* Este projeto foi desenvolvido com foco em design personalizado, experiência do usuário e compatibilidade com impressão de capas.

🎯 Objetivos
* Permitir que usuários criem capas únicas para seus celulares
* Oferecer ferramentas simples de edição gráfica
* Gerar visualização realista da capa
* Exportar o design pronto para produção ou compra

⚙️ Funcionalidades

👤 Usuário
* Criar conta
* Fazer login e logout
* Escolher modelo de celular
* Salvar designs personalizados
* Visualizar histórico de projetos

🎨 Editor de Capas
* Inserir imagens
* Adicionar textos
* Escolher cores e padrões
* Aplicar stickers
* Ajustar tamanho, rotação e posição dos elementos

✏️ Ferramenta de Desenho
* Desenho livre na capa
* Escolha de pincel
* Ajuste de espessura e cor

📱 Visualização
* Pré-visualização da capa
* Simulação no modelo de celular
* Zoom e rotação do design

🖼️ Exportação
* Exportar design em:
* PNG
* JPG
* PDF
* Resolução adequada para impressão (300 DPI)
